# Blue Ocean Plugin

Install marketplace:

```shell
/plugin marketplace add blairanderson/skills
```

Install Skill:

```
/plugin install blue-ocean@blairanderson-skills
```

This plugin adds one skill for applying Blue Ocean Strategy to any business, product, or career situation.

---

## `/blue-ocean`

Apply W. Chan Kim and Renée Mauborgne's Blue Ocean Strategy framework to stop competing and start creating uncontested market space.

Describe your business, product, service, or career situation — the skill delivers a full strategic analysis in seven steps:

1. **Red Ocean Diagnosis** — names the industry you're actually in and the factors every player competes on
2. **Value Curve Trap** — maps your current offering on the competitive landscape and identifies where you're racing to the middle
3. **Non-Customer Analysis** — finds the three tiers of non-customers and the biggest uncontested opportunity
4. **Four Actions Framework** — identifies what to Eliminate, Reduce, Raise, and Create to produce a divergent value curve
5. **New Value Curve** — prescribes the specific combination of moves that makes direct comparison to competitors almost impossible
6. **Blue Ocean Move** — one concrete action you can take in the next 30 days to begin testing the new curve
7. **Red Ocean Trap** — names the obvious move that feels like differentiation but is actually just another red ocean maneuver

Draws on the Strategy Canvas, Three Tiers of Non-Customers, Pioneer-Migrator-Settler Map, Buyer Utility Map, Blue Ocean Idea Index, and Tipping Point Leadership tools.
